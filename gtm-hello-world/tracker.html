Operator Quick Reference — What Do You Want to Do?

Find contacts for a campaign

Discover ICP-matched contacts from the enrichment database for a specific hospital, system, or segment.
/dh-contacts {id}
Also: /dh-accounts enterprise for batch, /dh-signals "{system}" for triggers

Launch a new campaign

Create campaign in Instantly, set email sequences, assign mailboxes, enroll contacts, activate sending.
/instantly-launch
Dry-run by default. Add --live for real API calls.

Add more contacts to a running campaign

Enroll new contacts into an existing active or paused campaign.
/instantly-enroll
Deduplicates against existing enrollment. Batches of 1,000.

Check how campaigns are performing

Pull latest analytics from Instantly, check open/reply/bounce rates, run kill-switch checks.
/instantly-metrics
Also: /instantly-status for full dashboard, /instantly-campaigns for list view

Process replies and classify sentiment

Pull new reply threads from Instantly, classify as positive/negative/neutral/OOO, triage next actions.
/instantly-replies
Keywords override AI scores. Positive replies flagged for Clay push.

Push positive replies to Salesforce

Send qualified positive replies to Clay webhook, which creates leads in Salesforce.
/instantly-clay-push
Only pushes unpushed positive replies. Logs to clay_push_log.

Pause or resume a campaign

Immediately stop or restart sending for a specific campaign.
/instantly-toggle
Use when kill-switch triggers or when you need to edit sequences.

Find a specific contact or account

Search by email, name, account name, ICP title, or department across local DB and Instantly.
/instantly-search
Auto-detects query type. Also: /instantly-account "HCA" for deep view.

Check mailbox health and warmup

Verify sending accounts are connected, warmed up, and not flagged.
/instantly-mailbox-health
Shows warmup score, daily limit, connection status per account.

Block a domain or email

Add to block list (DNC). Prevents future sends to that email or entire domain.
/instantly-dnc
Reversible — can list and remove entries. Local DB is source of truth.

Verify emails before sending

Check deliverability of email addresses before enrolling in a campaign.
/instantly-verify
5-45 seconds per email. Batch mode available from DB.

Sync everything from Instantly

Pull latest campaigns, leads, emails, accounts, and metrics into local DB.
/instantly-sync
5 sync jobs with cursor-based pagination. Idempotent — safe to re-run.

-- ICP-matched executives with email SELECT first_name, last_name, primary_email, title, hospital_name FROM executives WHERE icp_matched = 1 AND primary_email IS NOT NULL ORDER BY icp_intent_score DESC LIMIT 50; -- Recent leadership changes (R3 signals) SELECT e.first_name, e.last_name, e.title, ts.signal_type, ts.detected_date FROM executives e JOIN trigger_signals ts ON e.definitive_id = ts.definitive_id WHERE ts.signal_type IN ('new_hire', 'title_change') AND ts.detected_date >= date('now', '-6 months') ORDER BY ts.detected_date DESC; -- Negative-margin hospitals (R6 data) SELECT facility_name, metric_name, metric_value FROM account_enrichment WHERE data_type = 'financial' AND metric_name = 'operating_margin' AND CAST(metric_value AS REAL) < 0 ORDER BY CAST(metric_value AS REAL) ASC; -- ICP title distribution SELECT icp_title, COUNT(*) AS cnt FROM executives WHERE icp_matched = 1 GROUP BY icp_title ORDER BY cnt DESC;
